# 🚀 DivGrab

**DivGrab** is a lightweight Chrome extension that helps you **capture HTML, CSS, and JavaScript from any website by simply selecting elements**.

Built with the assistance of AI, it is designed to make frontend exploration faster, easier, and more practical.

---

## ✨ Features

* 🎯 Select any element on a webpage
* 📦 Extract **HTML structure**
* 🎨 Capture **CSS styles**
* ⚙️ Include related **JavaScript (if applicable)**
* ⚡ Fast and minimal workflow
* 🧠 Helpful for learning and reverse engineering UI

---

## 🧑‍💻 Use Cases

* Copy UI components for practice projects
* Save useful snippets for later use

---

## 🛠️ Installation

1. Clone or download this repository
2. Open Chrome and go to:
   `chrome://extensions/`
3. Enable **Developer Mode** (top right)
4. Click **Load unpacked**
5. Select the project folder

---

## 🚀 How to Use

1. Open any website
2. Click on the **DivGrab extension icon**
3. Activate selection mode
4. Click on any element you want to capture
5. View or copy the extracted HTML, CSS, and JS

---

## ⚠️ Limitations

* Dynamic JS-heavy components may not fully capture logic
* Some styles (like computed styles or external dependencies) may vary
* Works best with standard DOM elements
